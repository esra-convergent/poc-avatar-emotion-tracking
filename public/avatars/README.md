# VRM Avatars

Place your VRM avatar files here.

## Getting a VRM Avatar

### Option 1: Create Your Own (Recommended for POC)
1. Download **VRoid Studio** (free): https://vroid.com/en/studio
2. Create a character (takes 10-30 min for basic customization)
3. Export as `.vrm` file
4. Place the file here and name it `default-avatar.vrm`

### Option 2: Use Free VRM Models
- **VRoid Hub**: https://hub.vroid.com/en/models
  - Filter by "Free" license
  - Download `.vrm` file
- **Booth.pm**: https://booth.pm/en/search/VRM%20free

### Option 3: Sample Models for Testing
- **Alicia Solid**: https://hub.vroid.com/en/models/4470669254079636467
- Search "VRM free download" for more options

## File Naming
- Default avatar: `default-avatar.vrm`
- Custom avatars: any name, configure in app-config.ts

## File Size
- Recommended: < 10 MB for optimal performance
- VRM files are typically 2-10 MB
